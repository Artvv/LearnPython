История HTML

  • Создан Тимом Бернерсом-Ли
  • Создавался для обмена научными публикациями
  • Первая веб-страница создана 6 августа 1991 года
  • Стандартизирован Консорциумом Всемирной паутины (W3C)

История HTML

  • HyperText Markup Language
  • Ранние 90е
  • HTML 4 в 1997
  • XHTML в 2000
  • HTML 5 28 октября 2014

Определения

  • Веб-дизайн
    Проектирование пользовательских веб-интерфейсов для сайтов или веб-приложений.
  • Веб-разработка
    Процесс создания веб-сайта или веб-приложения.
  • Front end
    Разработка визуальных элементов веб-сайта или веб-приложения (на стороне клиента)
  • Back end
    Разработка функциональных элементов веб-сайта или веб-приложения (серверная сторона)

Как создать HTML документ?

  • В любом редакторе создать файл с расширением .html
  • Имя основного файла называем index.html
  • И начинаем писать HTML теги

Начало: Структура директории

Все файлы статического сайта должны находиться в одной директории.

Содержимое:

  • HTML файлы
  • CSS файлы
  • Изображения
  • Файлы JavaScript
  • Остоальные файлы, которые должны быть доступны на сайте

Примечание: Имена файлов не должны включать пробелы или специальные символы. Имена файлов чувствительны к регистру.

Анатомия HTML элемента

  • Элемент
    • Индивидуальный компонент HTML документа
    • Параграф, заголовок, таблица, список, div, ссылка, изобаражение, и т.д.
  • Тег
    • Отмечает начало & конец элемента
    • Открывающий тег и закрыващюий тег
    • Имя тага определяет его назначение
      • <tagname>Содержимое внутри</tagname>
    • <p>Это простой параграф.</p>

Анатомия HTML элемента

  • Элемент контейнер
    • Элемент, который содержит другой элемент

Анатомия HTML элемента

  • Атрибуты
    • Предоставляют дополнительную информацию о HTML элементе
    • Класс, идентификатор, язык, стиль, источник
    • Помещается внутри открытого тега перед закрыващюейся уголвой скопбкой.
  • Значение
    • Значение - это значения ассоциированные с данным атрибутом.
    • Значения должны размещаться в кавыках (одинарных или двойных).
    • <div id="copyright">© itstep.org 2016</div>

Doctype

Первая строка HTML страницы - это doctype (тип документа), который указывает браузеру на версию языка разметки.


          <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML
          4.01 Transitional//EN" "http://
          www.w3.org/TR/html4/loose.dtd">
        

          <!DOCTYPE html>

* doctype не чувствителен к регистру.
DOCtype, doctype, DocType и DoCtYpe являются допустимыми вариантами записи.

HTML тег

После <!doctype> все содержимое html страницы размещается между <html> тегами.


            <!DOCTYPE html>
            <html>

            </html>

head & body

Заголовок (head): Заголовок содержит заголовок страницы (title) & метаинформацию (meta) о странице. Матаинформация не видна пользователю, но служит важным целям. В частности, указывается автор и описание страницы, ключевые слова, что важно для поискового движка. Также указывается кодировка страницы и многое другое.

Тело (body): В тело включают содержимое страницы. Все, что находится внутри тега body видно пользователю.

Вложенность

Все элементы "вложены" один в другой

Например, наличие тега <p> внутри тега <body>. Тег <p> вложен в <body>

Вложенность: Пример

Элементы 'вложены' в тег <body>.


        	<body>
        	   <p>A paragraph inside the body tag</p>
        	</body>

Параграф 'вложен' в списочный элемент.


        	<ul>
        	   <li>
        	      <p>A paragraph inside a list item</p>
        	   </li>
        	</ul>

Ссылки имеют три компонента

Тег <a> может включать текст или картинки, преваращая их в ссылки

Атрибуты ссылок

Ссылки могут иметь атрибуты, которые указывают браузеру на различные действия, такие как открывать ссылку в новой вкладке или запускать почтовую программу.


<a href="home.html" target="_blank">Link Text</a>

Ссылка открывается в новом окне/вкладке с атрибутом target="_blank" 


<a href="mailto:shaptala@itstep.org">E-mail me!</a>

Ссылка открывает почтовую программу благодоря добавлению mailto: прямо перед email адресом.
